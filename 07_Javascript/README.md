**<h1>JAVASCRIPT<h1>**

Javascript adalah salah satu bahasa pemograman tingkat tinggi atau high-level, diartikan high-level karena semakin tinggi level bahasa semakin mudah dipahami oleh manusia.

adapun poin penting dari javascript :
1. declaration yaitu proses pembuatan variabel untuk menyimpan data

ada beberapa variabel yang harus diketahu yaitu var, let, const:
Var adalah jenis variabel yang dapat dideklarasikan ulang. var digunakan pada function scope dan global scope
let adalah jenis variabel yang nilai nya tidak tetap masih dapat diubah,namun let tidak dapat digunakan pada global scope
const adalah jenis variabel yang nilai nya tetap dapat diubah  dan juga const tidak dapat digunakan pada global scope

2. function adalah sebuah objek karena dalam function terdapat properti dan juga method. sebuah function dapat digunakan secara berulang kali.

3. DOM atau Document Object Model adlah sebuah API untuk HTML yang mempresentasikan halaman web pada suatu dokumen menjadi sebuah object
