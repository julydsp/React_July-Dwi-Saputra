**<h1>INTRODUCTION DATA STRUCTURE<h1>**

Data structure merupakan bagaimana menyimpan dan mengorganisir data/value/elemen di dalam memori komputer.
dengan kita belajar tentang data structure kita dapat membuat sebuah kode yang berkualitas dan efisien

terdapat basic data structure:
1.array
2.set

ada beberapa operasi dalam data structure seperti dibawah ini:

1. Read merupakan operasi yang bertujuan untuk membaca sebuah value. operasi read juga merupakan sebuah operasi tercepat dikarenakan hanya membutuhkan 1 step.
2. search merupakan sebuah operasi yang mencari suatu nilai tertentu. operasi search ini tidak memiliki cara untuk melompat langsung kenilai tertentu, dengan melakukan operasi search ini kita perlu memeriksa cell satu per satu
3. insert merupakan operasi untuk menambahkan suatu value keedalam array atau sebagainya. operasi insert ini dapat menjadi efisien tergantung dimana kita memasukan sebuah valuenya
4. delete merupakan operasi bertujuan menghapus suatu nilai pada indeks tertentu. kita dapat menghapus menggunakan sebuah method filter, menggunakan value, menggunakan splice,
