**<h1>BOOTSTRAP<h1>**

1. bootstrap merupakan salah satu sebuah framework untuk memudahkan dalam pembuatan tampilan website
2. class bootstrap ada beberapa :
    1.container
    2.row
    3.column
    4.button
3. di bootstrap kita juga dapat menggunakan grid. fungsi grid sendiri adalah untuk membagi layar menjadi beberapa col sesuai keinginan kita. untuk setiap layar memiliki col-12.

css position 
    1. absolute posisi dimana element akan naik satu dimensi tetapi ruangan yang ditinggalkan dianggap tidak ada
    2. relativ posisi dimana element naik satu dimensi tetapi ruangan yang ditinggalkan dianggap masih ada
    3. fixed posisi dimana element ketika kita gerakan top left right nya akan relatif terhadap windows browser
    4. static adalah nilai default dari suatu element ketika tidak diberi properti position
